# Multi-langue dans les Plugin


 - Utiliser la classe pour avoir toutes les langues configurées

    `Omega\Library\Language\Front\LangManager`

 - Multi-langue activé ?

    `Config::get('om_enable_front_langauge');`


 - Langue par défaut ?

    `Config::get('om_default_front_langauge')`
